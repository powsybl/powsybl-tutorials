# CGMES, merge and security analysis tutorial
In this tutorial, we learn how to import two CGMES files. The CGMES files are from two bordering countries: Belgium and
the Netherlands. These files are imported as two different networks and then merged in a single network. The second part
of this tutorial aims at computing a security analysis on this network.

# Running the tutorial
To run this tutorial, you need to execute the following command lines:
```
cd <path_to_powsybl_tutorials>/cgmes/
mvn compile exec:java
```
